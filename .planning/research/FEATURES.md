# Features Research

**Domain:** Crop Disease Detection Web App
**Researched:** 2026-03-20
**Confidence:** HIGH

## Feature Categories

### Table Stakes (Must Have)
- **Image Upload:** Camera capture or file upload for crop leaves.
- **Disease Classification:** CNN inference returning the top predicted disease.
- **Confidence Score:** Displaying how certain the model is.
- **Remedy/Advisory:** Text-based organic and chemical treatment suggestions.
- **Responsive UI:** Must work cleanly on mobile screens.

### Differentiators (Competitive Advantage)
- **Offline/PWA Mode:** Ability to cache the app shell for terrible internet.
- **Multilingual Support:** Hindi/Marathi localization for local Indian farmers.
- **History/History Tracking:** Saving past uploads to track farm health over time.
- **Weather Integration:** Alerting if current weather is highly conducive to blight/fungus.

### Anti-Features (What NOT to build)
- **Complex Social Feeds:** Distracts from the core diagnostic value.
- **Heavy 3D Graphics/Animations:** Drains battery and stalls low-end devices.
- **Mandatory Heavy Tutorials:** Needs to be intuitive enough to use immediately.

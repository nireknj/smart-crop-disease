# Plan 04-01: Backend Code Preparation for Production

## Objective

Prepare the backend code to be deployable on Render with a PostgreSQL database (Neon). This involves updating `database.py` to correctly handle `DATABASE_URL` for both SQLite (local) and PostgreSQL (prod), adding the `psycopg2-binary` driver, and ensuring the server binds to `$PORT`.

No new features — this is pure deployment plumbing.

---

## Tasks

### Task 1: Update backend/database.py for PostgreSQL compatibility

<read_first>
- `backend/database.py` — current SQLAlchemy setup and DATABASE_URL handling
</read_first>

<action>
Open `backend/database.py`. Apply the following changes:

1. Read DATABASE_URL from environment with SQLite fallback:
```python
import os
DATABASE_URL = os.environ.get("DATABASE_URL", "sqlite:///./sql_app.db")
```

2. Fix `postgres://` → `postgresql://` (Neon emits `postgres://`, SQLAlchemy requires `postgresql://`):
```python
if DATABASE_URL.startswith("postgres://"):
    DATABASE_URL = DATABASE_URL.replace("postgres://", "postgresql://", 1)
```

3. Use conditional `connect_args` — SQLite requires `{"check_same_thread": False}`, PostgreSQL does not:
```python
connect_args = {"check_same_thread": False} if DATABASE_URL.startswith("sqlite") else {}
```

4. Add connection pool limits for Neon's free tier (max 10 concurrent connections):
```python
engine = create_engine(
    DATABASE_URL,
    connect_args=connect_args,
    pool_size=5,
    max_overflow=0
)
```
Note: `pool_size` and `max_overflow` kwargs are not supported for SQLite dialects — guard them:
```python
if DATABASE_URL.startswith("sqlite"):
    engine = create_engine(DATABASE_URL, connect_args=connect_args)
else:
    engine = create_engine(DATABASE_URL, connect_args=connect_args, pool_size=5, max_overflow=0)
```

Full rewritten file content:
```python
import os
from sqlalchemy import create_engine
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker

DATABASE_URL = os.environ.get("DATABASE_URL", "sqlite:///./sql_app.db")

# Neon provides postgres://, SQLAlchemy requires postgresql://
if DATABASE_URL.startswith("postgres://"):
    DATABASE_URL = DATABASE_URL.replace("postgres://", "postgresql://", 1)

connect_args = {"check_same_thread": False} if DATABASE_URL.startswith("sqlite") else {}

if DATABASE_URL.startswith("sqlite"):
    engine = create_engine(DATABASE_URL, connect_args=connect_args)
else:
    engine = create_engine(DATABASE_URL, connect_args=connect_args, pool_size=5, max_overflow=0)

SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)
Base = declarative_base()
```
</action>

```python
import os
from logging.config import fileConfig

```
Then apply the above env.py edits.
